Making it easy to license your assets

디지털 자산을 적절히 라이선스하고 싶으신 마음, 잘 알고 있습니다.
VPLAY Elements 라이선스는 여러분이 올바른 선택을 쉽게 할 수 있도록 설계되었습니다.

Highlights

One simple commercial license

VPLAY Elements의 모든 항목은 동일한 간단한 라이선스 조건을 따릅니다. 넓은 상업적 사용 권한을 제공하여, 업무나 개인 프로젝트에서 자신 있게 사용할 수 있습니다.

Register items each time you use them

항목을 다운로드할 때마다 특정 용도로 지속적인 라이선스를 부여받습니다. 동일 항목을 다른 용도로 다시 사용하고 싶다면, 다운로드 기록에서 간단히 재등록하면 됩니다.

Clear guidelines on what's not allowed

다음과 같은 상식적인 제한이 적용됩니다 :


  • 항목을 재판매하거나 재배포할 수 없습니다.
  • 주문형 서비스(on-demand services)에서 항목을 사용할 수 없습니다.
  • 항목을 상품화의 기초로 사용할 수 없습니다. 예를 들어, VPLAY Elements의 로고를 그대로 티셔츠에 인쇄하는 것은 허용되지 않습니다.
  • 음악 항목은 방송 프레젠테이션 에서 사용할 수 없습니다.

Future proof licensing, even if you unsubscribe

구독이 종료되면 VPLAY Elements의 항목을 더 이상 사용할 수 없지만, 이미 등록된 기존 사용은 계속 보장됩니다!
